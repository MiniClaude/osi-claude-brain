# Skills

Store your custom Claude skill files here.

When Anthropic updates the Cowork plugin, your custom skills live safely in this repo — not in a temp folder that gets wiped.

## How to add a skill

1. Copy your skill `.md` file into this folder
2. Run `scripts/push.bat` to save it to GitHub
3. On your other computers, run `scripts/pull.bat` to get it
