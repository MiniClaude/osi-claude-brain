# OSI Claude Brain 🧠

Brian Charrette's Claude automation hub — skills, email sequences, and scheduled tasks, synced across all computers via GitHub.

## What's in here

| Folder | What it stores |
|--------|---------------|
| `skills/` | Custom Claude skill files (.md) |
| `automation/` | Email queue, task configs, sequence templates |
| `scripts/` | One-click batch files (push, pull, sync) |
| `docs/` | Git cheat sheet and guides |

## Syncing across computers

**Every time you make changes:**
1. Double-click `scripts/push.bat` → saves your changes to GitHub

**When you sit down at a different computer:**
1. Double-click `scripts/pull.bat` → grabs the latest from GitHub

That's it. Two clicks, always in sync.
